# Developers workplace data management system

This project is a SQLAlchemy ORM designed to manage employees data - personal info, role and experience. The goal of this project is to show practical usage of object oriented programming fundamentals in real world academic context.

# Technologies
- Python
- SQLAlchemy

## Concepts
- Object-Oriented Programming (OOP)
- ORM Mapping (Object Relational Mapping)
- Database Relationships
